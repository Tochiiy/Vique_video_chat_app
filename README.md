# Vique - Real-Time Video Chat Application MVP PROJECT

Vique is a fully featured, real-time video chat application built using the MERN stack (MongoDB, Express, React, Node.js) combined with WebRTC and Socket.io to facilitate seamless, low-latency video and audio communication directly within the browser.

🔗 **Live Demo:** [vique-video-chat-app.onrender.com](https://vique-video-chat-app.onrender.com/)

---

## 🚀 Features 

- **Real-Time Video & Audio:** Peer-to-peer high-quality streaming using WebRTC.
- **Instant Messaging:** text chat overlay during video calls powered by WebSockets.
- **Room Creation & Joining:** Generate unique room links to easily invite others.
